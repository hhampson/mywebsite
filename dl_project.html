Drought Prediction in the Western U.S. using a Long Short-Term Memory (LSTM) Model

january - march 2021

Project Summary

We used a Long Short-Term Memory (LSTM) model to predict drought index values across the Western U.S. Droughts are a large threat to this region of the U.S., and are predicted to worsen under future climate conditions. The model predicts the severity of droughts using input measurements of precipitation, temperature, and soil moisture from the wetter 6-month season, outputting a 6-month averaged drought index forecast for the following dry season. We were able to obtain data from 3 years, 2015-2018.

This project was undertaken as part of Computer Science 230: Deep Learning at Stanford University.

Results

A basic linear regression model gave a test set error of 0.137 and our LSTM model gave a test set error of 0.0699 - a promising improvement!
